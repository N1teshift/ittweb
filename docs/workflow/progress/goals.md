# Project Goals and Progress Tracking

This file tracks high-level goals and their progress. Tasks in `agent-tasks.md` should link to goals here.

## Goal Format

```markdown
## [Goal Name]

**Status**: Not Started | In Progress | Blocked | Complete
**Progress**: X% (updated: YYYY-MM-DD)
**Target Date**: YYYY-MM-DD (optional)
**Success Criteria**:
- Criterion 1
- Criterion 2

**Related Tasks**: [Links to tasks in agent-tasks.md]
**Assigned Agents**: [List of agents working on this]

**Notes**: Any important context or blockers
```
