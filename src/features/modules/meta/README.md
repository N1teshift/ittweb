# Meta Module

**Purpose**: Analytics dashboard displaying system-wide statistics and charts.

## Exports

### Components
- `MetaPage` - Main analytics dashboard page

## Usage

The meta page is accessible at `/meta` and displays:
- Activity charts
- ELO statistics
- Win rate data
- Class statistics
- Player activity

## API Routes

- `GET /api/analytics/meta` - Get meta dashboard data
